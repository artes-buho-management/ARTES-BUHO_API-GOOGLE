# Catalogo de APIs

Registro de APIs de Google usadas en el proyecto.

## Previstas

### Google Places API
- Uso: busqueda de empresas/lugares para proyectos de discovery.
- Proyectos consumidores: APP_ARTES-BUHO_BUSCA-CONTACTOS, APP_ARTES-BUHO_DISTRITOS-MADRID.
- Variable: `GOOGLE_PLACES_API_KEY`.
- Estado: PENDIENTE alta.

### Google Sheets API
- Uso: lectura/escritura de hojas de control y CRM.
- Estado: PENDIENTE.

### Google Drive API
- Uso: gestion documental y copias de seguridad.
- Proyecto consumidor: ARTES-BUHO_RAMON (backend/app/tasks/drive_crm_backup.py + drive_organizer.py).
- Estado: PENDIENTE.

### Gmail API
- Uso: envio y lectura de correo transaccional.
- Proyecto consumidor: ARTES-BUHO_GMAIL.
- Estado: PENDIENTE.

### Google Calendar API
- Uso: gestion de eventos y reuniones.
- Estado: PENDIENTE.

### OAuth2 / Service Accounts
- Uso: autenticacion para APIs que requieren usuario o cuenta de servicio.
- Estado: PENDIENTE.
